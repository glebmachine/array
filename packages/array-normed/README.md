# array-normed

[![NPM version][npm-image]][npm-url]
[![npm download][download-image]][download-url]

Get the average value in an array.

## Installation

`$ npm install --save ml-array-normed`

## Usage

```js
import normed from 'ml-array-normed';

const result = normed([1, 2, 3, 4]);
// [0.1, 0.2, 0.3, 0.4]
```
